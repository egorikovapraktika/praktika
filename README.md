## Руководство по установке OpenMediaVault

Полная версия находится в папке `openmediavault_guide/`.  
Краткая последовательность:

1. Скачать ISO OpenMediaVault с официального сайта.
2. Создать виртуальную машину в VirtualBox (Linux/Debian 64-bit, 2 ГБ RAM, 20 ГБ диск).
3. Настроить сетевой адаптер в режиме «Сетевой мост».
4. Загрузиться с ISO, выполнить установку.
5. После перезагрузки получить IP-адрес и войти в веб-интерфейс (`admin` / `openmediavault`).
6. Добавить и инициализировать дополнительный диск для данных.
